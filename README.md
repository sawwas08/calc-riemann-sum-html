# calc-riemann-sum-html
a little script for calculating an approximate area between x axis and curve for a function

This script is extremely hardcoded for the function used, and currently only has support for midpoint riemann sum. The output for point pairs is not implemented. logic is well organized and straightforward. committed as of 4/13/~10:38pm

HOW TO USE:
download the html file in the repository, and either open it in a browser with the filepath to the download on your machine, or simply drag the file into a browser url bar.
